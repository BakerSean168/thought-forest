# SYSFAN-header 

![[SYSFAN-header.png]]

这是一个 **SYSFAN1 接口**，也就是 **系统风扇/机箱风扇接口**。

### 1. 核心身份

- **丝印**: `SYSFAN1` (System Fan 1)。
- **类型**: **标准 4-Pin PWM 风扇接口**。
- **位置**: 通常位于主板边缘，方便连接机箱后部或前部的散热风扇。

### 2. 它是做什么的？

- **给机箱散热**: 这个接口不是给 CPU 风扇用的（CPU 风扇通常插在 `CPU_FAN` 上），而是专门用来接 **机箱上的辅助风扇**。
- **智能调速 (PWM)**: 因为它是 4 针的，意味着它支持 **PWM 脉宽调制**。
	- **功能**: 主板可以根据系统温度自动调节风扇转速。温度低时慢转（静音），温度高时快转（散热）。

### 3. 针脚定义 (标准 4 针)

这个接口遵循 PC 工业标准：

1. **GND**: 地线
2. **+12V**: 电源 (供电)
3. **Sense**: 测速 (告诉主板风扇转得有多快)
4. **Control**: PWM 信号 (告诉风扇该转多快)

### 4. 给你的 N100 软路由/NAS 建议

- **必须接风扇吗？**
	- 如果你是用这块板子做 **软路由**（只跑流量，不装机械硬盘），N100 发热很小，可能不需要接额外的机箱风扇。
	- **但是**，如果你是用它做 **NAS**（装了 2.5寸 或 3.5寸 机械硬盘），**强烈建议在这里接一个机箱风扇**对着硬盘吹。硬盘长期高温（超过 45°C）会大大缩短寿命。
- **3 针风扇能插吗？**
	- **可以**。如果你手头只有普通的 3 针风扇，直接插进去也是能转的（卡扣位置是兼容的，只占用前 3 针），只是可能会失去 PWM 智能调速功能，一直全速转或者只能通过电压调速（取决于 BIOS 支持）。
- **噪音控制**:
	- 如果你觉得风扇太吵，可以在 BIOS 的 `Hardware Monitor` 或 `Smart Fan` 选项里，针对 `SYSFAN1` 设置静音模式 (Silent Mode)。

**总结**: 这是给机箱风扇供电和调速的接口。如果是做 NAS，请务必利用好它来给硬盘散热。
