# 嵌入式

### 技术方向：必须会的

1. C 语言（底层单片机编程）
2. Altium Designer 画图（设计电路板）
3. solidworks（设计结构）
4. 射频部分（设计简单天线）
5. C#（很多硬件需要上位机）

### 精通

1. usb 技术（hid， usb mouse， usb keyboard， u盘）
2. can 技术
3. uart 技术
4. 单片机 flash
