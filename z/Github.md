# Github 

## 概览（Overview）

* 一个代码托管平台
* 一个开源代码平台

## 学生优惠

[[GitHub学生认证]]







github 镜像：

|镜像站点名称|访问地址|主要特点|适用场景|
|---|---|---|---|
|​**​bgithub.xyz​**​|[https://bgithub.xyz/](https://bgithub.xyz/)|直接替换域名访问，操作简单|日常浏览仓库、克隆代码|
|​**​kkgithub.com​**​|[https://kkgithub.com/](https://kkgithub.com/)|直接替换域名，支持代码查看和 Issue|日常浏览仓库、查看 Issues|
|​**​gitclone.com​**​|[https://gitclone.com/](https://gitclone.com/)|提供在线工具生成克隆命令，适合命令行操作|需要快速获取仓库克隆命令|
|​**​kgithub.com​**​|[https://kgithub.com/](https://kgithub.com/)|支持代码查看、Issue 和评论，但不支持注册和文件上传|阅读代码、参与讨论（无需上传文件）|
|​**​ghproxy.net​**​|[https://ghproxy.net/](https://ghproxy.net/)|主要用于​**​加速文件下载​**​，通过在原链接前添加代理前缀实现|加速 Releases、源码压缩包、大文件下载|
|​**[​moeyy.cn/gh-proxy​](http://xn--moeyy-kt3b.cn/gh-proxy%E2%80%8B)**​|[https://moeyy.cn/gh-proxy/](https://moeyy.cn/gh-proxy/)|同样专注于文件下载加速，用法类似 [ghproxy.net](http://ghproxy.net/)|加速 Releases、源码压缩包、大文件下载|
|​**​Gitee (码云)​**​|[https://gitee.com/](https://gitee.com/)|​**​国内代码托管平台​**​，支持手动或自动同步 GitHub 仓库|需要稳定、快速的代码托管和协作（尤其是国内团队）|
|​**​GitCode​**​|[https://gitcode.com/](https://gitcode.com/)|类似 Gitee 的国内代码托管平台，也可镜像 GitHub 仓库|需要稳定、快速的代码托管和协作（尤其是国内团队）|
