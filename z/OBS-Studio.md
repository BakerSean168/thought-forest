# OBS Studio 

它是全球最受欢迎的直播工具，支持将视频内容推送到 Bilibili、YouTube、Twitch 等各大主流平台。软件允许用户通过“场景”和“来源”的自由组合，实时捕捉屏幕、窗口、摄像头或特定的音视频采集卡信号，并进行复杂的混音与滤镜处理。

- **核心功能：** 支持实时视频渲染与编码，具备多路来源混合、转场动画以及高清本地录制功能。
- **技术亮点：** 采用高性能的 GPU 加速渲染技术，拥有极具扩展性的插件系统（如 OBS-WebSocket）和强大的虚拟摄像头功能。
- **适用场景：** 广泛应用于游戏直播、网课录制、视频会议演示以及专业级的多机位现场导播。
