# 无线网卡接口 

![[M.2-WIFI-header.png]]

这是一个 **M.2 WiFi 接口 (M.2 E-Key Slot)**。

在你的 N100 主板上，这个接口的主要用途是安装 **无线网卡**。

### 1. 核心参数

- **丝印标记**: 接口下方清晰地印着 **M.2 WIFI**。
- **接口类型**: **M.2 E-Key**（注意它的防呆缺口位置，和你的 M.2 NVMe 硬盘插槽是不一样的，插不进去硬盘）。
- **信号协议**: 这个接口通常包含两组信号：
	- **PCIe x1**: 用于传输 WiFi 数据（速度较快）。
	- **USB 2.0**: 用于传输 蓝牙 (Bluetooth) 协议。

### 2. 标准用法

- **装无线网卡**: 你可以买一张笔记本用的无线网卡（比如 Intel AX210, BE200, 或者便宜的 AC3168）插在这里。
- **用途**: 让你的服务器拥有 WiFi 功能，或者利用 Linux/OpenWrt 把这台机器变成一个无线路由器 (AP)。

### 3. 进阶玩法 (Home Lab / 软路由玩家必看)

对于做 PVE / 软路由的你来说，WiFi 功能可能很鸡肋（服务器通常用有线）。但这个接口其实是一个 **PCIe x1 扩展位**，可以通过转接卡玩出花来：

1. **转接 2.5G 网口 (推荐)**:
	
	- 淘宝上有卖 **"M.2 E-Key 转 RJ45"** 的转接卡。
	- 插上后，你的 N100 就能多出一个 2.5G 网口。如果你的主板原本只有 2 个网口，这样就能变成 3 网口，非常适合做 All-in-One（直通一个给软路由，剩下的做管理口）。
		
2. **转接硬盘 (SATA)**:
	
	- 可以买 **"M.2 E-Key 转 SATA"** 的转接卡，扩展出 2 个 SATA 接口接机械硬盘。
		
3. **安装 AI 算力卡 (Google Coral)**:
	
	- 如果你玩 Home Assistant 或 Frigate（监控 AI 识别），可以买 **Google Coral Edge TPU (M.2 E-Key 版)** 插在这里，专门用来跑 AI 模型，让 N100 的 CPU 腾出手来干别的。

### 4. 安装注意事项

- **螺丝柱**: 接口下方有一个银色的螺丝柱，用来固定网卡尾部。
- **天线**: 如果你装了无线网卡，别忘了还需要买 **IPEX4 转 SMA 的馈线** 和 **天线**，并把天线引到机箱外面，否则信号基本为零。

**总结**：这是给无线网卡留的位置。如果你不需要 WiFi，强烈建议买个 **M.2 A+E Key 转 2.5G 网口** 的转接卡，几十块钱就能给你的软路由增加一个物理网口，性价比极高。
