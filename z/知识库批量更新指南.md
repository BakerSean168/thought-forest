# 知识库批量更新指南

**上级索引**：[[000_Home]] | [[知识库扁平化管理]]

## 概述

本文档提供了将现有知识库文档批量更新为扁平化管理格式的指南。

## Frontmatter 模板

每个文档应包含以下 frontmatter：

```yaml
---
tags:
  - [领域标签]  # 如 dev/vue, ai/mcp, cybersec/ctf
  - [类型标签]  # 如 type/concept, type/howto, type/howto
  - [状态标签]  # 可选：status/seed, status/growing, status/evergreen
description: [简短描述]
created: [创建时间]
updated: [更新时间]
---
```

## 命名建议

- 使用描述性文件名：`Vue3 响应式原理.md` 而非 `响应式.md`
- MOC 文件统一后缀：`XXX MOC.md`
- 避免使用特殊字符和空格（可用连字符或下划线）

## 批量处理建议

对于大量文档，可以使用以下策略：

1. **分批处理**：按文件夹/领域逐批处理
2. **优先级排序**：先处理常用文档
3. **使用 Obsidian 搜索**：按标签搜索，批量更新
