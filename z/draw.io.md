# draw.io 

**draw.io**（现官方更名为 **diagrams.net**，但大家习惯仍叫它 draw.io）是一款**完全免费、开源的在线绘图工具**。

它是目前市面上**Microsoft Visio 最好的免费替代品**，深受程序员、产品经理和学生党的喜爱。

以下是它的核心介绍：

### 1. 核心优势

* **完全免费且无需注册：** 这是它最大的卖点。打开网页（draw.io）就能直接开始画图，不需要注册账号，没有任何付费解锁的高级功能，也没有水印。
* **隐私与存储自由：** 与其他强制你把文件存在他们服务器上的工具不同，draw.io **不存储你的数据**。
* 你可以选择把文件存在**本地电脑**。
* 也可以直接关联存到 **Google Drive, OneDrive, GitHub** 或 **GitLab** 中。
* 这意味着你的商业机密非常安全，完全由你自己掌控。



### 2. 功能强大

尽管是网页版工具，它的功能却毫不含糊：

* **绘图类型全：** 支持流程图、UML 图（程序员用）、网络拓扑图、ER 图（数据库设计）、组织结构图，甚至还有简单的平面设计图。
* **Visio 完美兼容：** 它可以**导入和编辑** Visio (.vsdx) 文件。如果你没有买昂贵的 Visio，用它来救急查看或修改别人的文件非常方便。
* **丰富的素材库：** 内置了 AWS、Azure、Google Cloud 的官方图标库，画云架构图非常专业。

### 3. 多平台支持

* **Web 端：** 任何浏览器打开即用。
* **桌面版：** 提供 Windows、macOS 和 Linux 的离线客户端，在没有网络的时候也能画图。
* **集成：** 它是 Confluence 和 Jira 中最流行的绘图插件之一。

### 总结

**draw.io** 是**画流程图的首选工具**。
它虽然没有 ProcessOn 那么多社交模板，也没有 Miro 那么炫酷的协作白板功能，但它**纯粹、免费、保护隐私且专业**。只要你不想花钱买 Visio，选它绝对没错。
