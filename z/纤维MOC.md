# 纤维MOC 

## 常用纤维

### 天然纤维

棉  

- 长绒棉 36mm
- 细绒棉 28mm
- 精梳棉 把短棉花梳走
- 丝光棉 棉花放入烧碱，横截面变圆
- 彩棉

麻  

总的来说，现在生产的麻面料已经比较舒服了。亚麻是欧洲一直很受欢迎的夏季面料，纯亚麻的价格和一些比较高级的棉花差不多，当然亚麻也分等级。大麻在古代确实是很粗糙的，但是现在我国已经掌握了处理大麻的工艺，纺出的布和亚麻差距不是很大，雅戈尔的汉麻也是大麻。苎麻（夏布）也是一种自古以来的面料，三种面料的衣服我都买过，总的来说，大麻和亚麻比较类似，像布。苎麻摸起来有点滑，整体又有点粗糙，感觉有点纸那种质感，热天穿薄苎麻衣服的感觉很不错。三种材料都有透气很好，遇水不会变透明，容易干的优点，也都易皱。黄麻就不了解了

### 化学纤维

再生纤维素纤维  

- 粘胶纤维：“竹纤维”，棉的再生纤维，吸湿性胜棉50%；  
- 莫代尔纤维：贴身内衣内裤；  
- 莱赛尔纤维：更高级，常被人称为天丝(×)—天丝是商标，包括以上两个尔纤维；  
- 醋酯纤维：吸附焦油性好，质感高级  

合成纤维  

- 涤纶：常见涤纶PET，还有PTT，PBT，阳离子（其因“阳”字，使得部分商家诈骗宣传像太阳一样发热）；涤纶并非由塑料瓶而来，而是直接化工合成，塑料瓶做成的会标识“再生聚酯纤维”  
- 锦纶：又叫尼龙，耐磨性好（冲锋衣、羽绒服、防晒服），价格贵涤纶三倍，湿水时更坚韧（渔网）  
- 腈纶：外号人造羊毛，因而见于冬季毛衣内衣。德绒：德国腈纶商标；美雅迪—日本。腈纶不发热，发热是粘胶纤维吸湿发热，保暖得看厚度和空气储存量  
- 氨纶：有弹力，给面料加弹力，遇84则不弹；莱卡：氨纶。  


## 纺织品处理工艺 (Finishing & Construction)

### 1. 织造工艺 (Construction)

决定面料的**厚度、弹力与透气性**。

- **无缝一体织 (Seamless)：**
	- **原理：** 像织袜子一样直接织出一件衣服，侧边没有缝合线。
	- **特点：** 减少摩擦，外观平整（休闲感强），能根据身体部位调节疏密（如腋下织得更稀疏）。
	- **代表：** 你选的 KIPRUN RUN 500。
- **华夫格/方格织法 (Grid Fleece)：**
	- **原理：** 面料内侧呈一个个小方格状凸起。
	- **特点：** 凸起部分保暖，凹陷部分排汗，是**高强度锻炼**的顶级中层选择。
- **双面织法 (Double Knit)：**
	- **原理：** 内外两层材质不同（如内锦纶外棉）。
	- **特点：** 兼顾舒适与功能，常见于“棉感”运动衫。

### 2. 表面整理 (Surface Finishing)

决定面料的**触感与初始温感**。

- **抓绒/起绒 (Napping/Brushing)-：**
	- **原理：** 用钢针或刺果将纤维拉出表面。
	- **特点：** 增加“空气储存量”，提高保暖力。**抓绒衣**的主要工艺。
- **磨毛 (Sanding/Peaching)：**
	- **原理：** 用砂纸磨。
	- **特点：** 形成像蜜桃皮一样的短绒感，触感温润，**日常打底衫**常用，减弱化纤的“塑料感”。
- **丝光 (Mercerization)：**
	- **原理：** 浓碱处理（你笔记中棉花那部分已记录）。
	- **特点：** 消除毛羽，使表面像丝绸一样光滑，**不容易起球**。

### 3. 功能性整理 (Functional Finishing)

给衣服加“特殊技能”。

- **DWR 防泼水处理：**
	- **原理：** 在面料表面形成“荷叶效应”。
	- **特点：** 应对南方小雨和湿气。注意：这**不是防水**，洗多了会失效。
- **银离子防臭 (Anti-odor)：**
	- **原理：** 面料中嵌入银离子或喷涂涂层。
	- **特点：** 抑制细菌生长，适合你提到的**高强度锻炼出汗**，防止化纤衣服发臭。
- **防静电处理：**
	- **特点：** 解决涤纶、腈纶在干燥季节容易粘身和“噼里啪啦”的问题。
