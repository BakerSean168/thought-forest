# PAN_PWR_header 

![[PAN_PWR_header.png]]

这是一个 **屏幕供电电压选择跳线 (Panel Power Voltage Selection)**。

这也是一个**极其危险**的跳线（如果你接了内置屏幕的话），但在你的家庭服务器用途下，它是**完全无用**的。

### 1. 核心身份

- **丝印**: `PAN_PWR1` (Panel Power 1)。
- **功能**: 它的作用是选择主板给 LVDS/eDP 液晶屏幕接口供电的电压。
- **常见档位**: 通常这三个针脚分别代表 **3.3V**、**5V** 和 **12V**。通过跳线帽短接不同的针脚，来改变输出电压。

### 2. 为什么会有这个东西？

因为这块板子是为 **一体机 (All-in-One PC)** 设计的。

- 有些一体机的屏幕面板需要 3.3V 驱动。
- 有些大尺寸面板需要 5V 或 12V 驱动。
- 工厂组装时，工人们会根据具体使用的屏幕型号来插这个跳线。

### 3. “甚至会炸”的风险

**千万不要手欠去拔插它。**

- **场景**: 假设你未来心血来潮，搞了一块笔记本拆机屏想接到这块主板的 LVDS 接口上。
- **后果**: 如果你的屏幕只需要 3.3V，但这个跳线误接到了 12V 档位，通电的一瞬间，你的**屏幕驱动板和背光电路会直接烧毁**，伴随青烟和焦味。

### 4. 给你的最终建议

**彻底无视它，保持现状。**

- 你使用的是 N100 做软路由或服务器，连接显示器用的是外部的 **HDMI** 或 **DP** 接口。
- 外部接口的电压是标准化的，**不受这个跳线影响**。
- 这个跳线只控制主板内部那排黑色 LVDS 针座的供电。只要你不往 LVDS 接口插线，这个跳线无论设成 3V 还是 100V 都对你没有任何影响。**让那个黑色跳线帽留在原地别动就好。**
