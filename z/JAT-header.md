# JAT-header 

![[JAT-header.png]]

这是一个非常有价值的跳线，特别是对于你做 **家庭服务器 (Home Server)** 或 **软路由** 来说。

这是 **AT/ATX 模式切换跳线**，俗称 **“来电自启”跳线**。

### 1. 核心身份

- **丝印**: `JAT1` (Jumper AT).
- **含义**: 选择电源的工作模式是 **AT 模式** 还是 **ATX 模式**。
- **物理结构**: 3 针跳线，目前跳线帽插在其中两针上。

### 2. 它是做什么的？

它决定了当你把电源适配器插到主板上（或者家里停电又来电）时，主板会怎么反应：

- **ATX 模式 (默认)**:
	- 通电后，主板处于 **待机状态**。
	- 你需要**手动按一下开机键**，电脑才会启动。
	- 这是普通家用电脑和笔记本的逻辑。
- **AT 模式 (服务器模式/来电自启)**:
	- 通电后，主板 **立刻自动开机**。
	- 不需要按任何按钮。
	- 这是广告机、工控机、路由器的逻辑。

### 3. 对你的 PVE 服务器有什么用？

**这是个神器。**

对于 24 小时运行的服务器或软路由，最怕的就是家里停电。

- 如果设置在 **ATX 模式**：停电再来电后，服务器是关着的，你必须跑回家按开关，或者家里断网没法远程。
- 如果设置在 **AT 模式**：来电瞬间，服务器自动启动，PVE 自动引导，网络自动恢复。

### 4. 如何使用？

你可以看到它是 3 根针，跳线帽目前短接了其中两根（假设是 1-2）。

如果你想要 开启“来电自启”功能，操作如下：

1. **断电**。
2. 拔下那个黑色的跳线帽。
3. 把它插到 **另外两根针** 上（比如从 1-2 移到 2-3）。
4. **通电测试**：插上电源线，不要按开关，看风扇转不转。如果直接转了，恭喜你，硬改“来电自启”成功。

**注意**：很多 N100 主板在 BIOS 的 `Chipset` -> `PCH-IO Configuration` -> `State After G3` 里也能设置来电自启。但 **JAT1 跳线是物理硬件层面的强制覆盖**，比 BIOS 设置更稳定（不会因为 BIOS 重置而失效）。

**总结**：这是控制 **“通电自动开机”** 的物理开关。建议你把它调成 **AT 模式**（即通电即开机），这样你的软路由在无人值守时会更可靠。
