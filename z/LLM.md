# LLM

**上级索引**：[[AI MOC]]

大语言模型，Large Language Model，LLM

## 基础概念

用 特殊方法 和 大量数据 训练生成的 模型  
模型能够根据输入的内容来**推断**接下来的内容  

不是所有的牛奶都叫_____。  
宫廷玉液酒，_____。

## 不能做什么

1. 不能进行逻辑推理
2. 没有上下文（记忆）
3. 不能运行代码、打开浏览器（行为）
