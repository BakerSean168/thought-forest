## 基础概念

ShadCN 最初是为 React 生态系统设计的 UI 组件库，因其美观的设计和高度可定制性而广受欢迎。ShadCN/Vue 是将这一设计理念和组件实现移植到 Vue 3 的项目。

### 特点

- 美观的默认设计
- 良好的开发体验
- 活跃的社区支持
- 详细的文档
- 无障碍支持好，定制能力强
- 相对较新的项目，可能不如一些成熟 UI 库稳定
- 组件数量可能不如一些大型 UI 库丰富
- 主要面向 Vue 3，对 Vue 2 支持有限
