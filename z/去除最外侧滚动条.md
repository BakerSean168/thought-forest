# 让 app （body） 右侧滚动条消失

在 html 中添加
```css
<style>
  body {
    /* Remove explicit overflow setting */
    -ms-overflow-style: none;  /* IE and Edge */
    scrollbar-width: none;     /* Firefox */
  }
  
  /* 只添加下方代码正确生效，添加上方代码就仍会有滚动条 */
  body::-webkit-scrollbar {
    display: none;  /* Chrome, Safari and Opera */
  }
</style>
```
