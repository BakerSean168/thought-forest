# **Sumatra PDF**



它是全球公认的最简洁的阅读器之一，以其近乎瞬时的启动速度和极低的资源占用而闻名。不同于体积庞大的同类软件，Sumatra PDF 的设计理念是“极简主义”，去除了一切不必要的冗余功能（如复杂的编辑或批注工具），专注于提供最纯粹、最流畅的阅读体验。

- **核心功能：** 支持单页、双页、书籍模式预览，具备强大的关键词搜索、书签管理及命令行支持。
- **技术亮点：** 基于高性能的 MuPDF 渲染引擎，单个可执行文件即可运行（无外部依赖），且在阅读时不会锁定 PDF 文件，方便开发者在更新文件后直接通过 `R` 键刷新查看效果。
- **全格式支持：** * **文档：** PDF, XPS, OpenXPS, DjVu, CHM。
	- **电子书：** EPUB, MOBI, FB2, PRC。
	- **漫画：** CBZ, CBR, CBT, CB7。
- **适用场景：** 适用于追求系统简洁的用户、低配置电脑办公、电子书爱好者以及需要高频率刷新文档的 LaTeX/技术文档开发者。
