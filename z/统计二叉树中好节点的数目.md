# 统计二叉树中好节点的数目 

## 题目

[1448. 统计二叉树中好节点的数目 - 力扣（LeetCode）](https://leetcode.cn/problems/count-good-nodes-in-binary-tree/description/)

## 题解

### [[链表-二叉树-回溯#自顶向下 DFS（先序遍历）]]

```js
var goodNodes = function(root) {
    let res = 0;
    if (!root) return 0;
    // 把 当前路线上的最大值传下去
    function dfs(node, max) {
        if (!node) return;
        if (node.val >= max) {
            res++;
        }
        max = Math.max(max, node.val);
        dfs(node.left,max);
        dfs(node.right,max)
    }
    dfs(root, -Infinity)
    return res;
};
```
