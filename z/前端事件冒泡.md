## 实战经验

实现一个提醒页面，有 ReminderTemplate ReminderGroup 和 桌面。
希望实现右键桌面出现创建菜单，右键对应 item 时，出现对应的菜单；
但是一开始实现时只能出现桌面的菜单

猜测是事件冒泡的原因,果然
```ts
/**
 * 处理分组右键菜单
 */
const handleGroupContextMenu = (group: ReminderTemplateGroup, event: MouseEvent) => {
  event.stopPropagation(); // 阻止事件冒泡到桌面
  contextMenu.x = event.clientX;
  contextMenu.y = event.clientY;
  contextMenu.items = [
    {
      title: '查看分组',
      icon: 'mdi-folder-open',
      iconColor: 'primary',
      action: () => {
        showGroupTemplates(group);
        contextMenu.show = false;
      },
    },
}
```
