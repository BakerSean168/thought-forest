# 路由 

## 基础概念

路由是怎么匹配的

路由在以前是用来访问特定的静态文件，渲染不同的页面。
现在有了 ajax、axios 等异步请求技术，能够实现页面不刷新也向服务器获取新数据，理论上很多页面可以没有路由，但应该考虑路由的更多作用，去维持良好的路由。接下来以我在实践为例。
### 路由的好处

| 操作     | 当前方式 | 改进方式        |
| ------ | ---- | ----------- |
| 保存会话状态 | 仅内存  | 内存 + URL 参数 |
| 刷新页面后  | 丢失选择 | 恢复到刷新前的会话   |
| 分享会话链接 | 不可行  | 可直接分享会话链接   |
| 浏览器回退  | 无法工作 | 支持浏览器历史     |
