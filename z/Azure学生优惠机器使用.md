# Azure学生优惠

## 概览（Overview）

Azure 是 Microsoft 的云平台，在学生认证后，可以使用 Azure 的免费服务（100$/年）。  


**学生优惠获取**：
- Azure的学生优惠可以直接在Azure的官网的学生优惠申请中进行填写。 
- 好像也可以直接通过经过了GitHub学生包申请后的GitHub账号登录进行获取。 

免费资源：
- 1台 Windows 和 1台 Linux 虚拟机 
- 2v1g 或 1v1g 配置
- 64G 磁盘

## 实战经验

### 语言改为英文

我经历到的一个问题是，当页面是非英文的时候，也就是中文的时候，会在创建机器时报各种奇怪的错误，比如说资源违控、资源已创建或者被禁止。

后面改成了把页面的语言改成英文，然后就没有这个问题了。不清楚具体的原因。 

### 费用开销

剩余的100刀每年是不够升级免费资源的配置的。 

这笔钱只能够用于短时间开一台高配置机器。 

免费资源没有任何免费的流量额度，使用的流量费用需要从每年的100刀中扣除。
