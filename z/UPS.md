# UPS 

你这个问题问到了 **All-in-One (AiO) 系统的灵魂**。对于像你这样打算把拨号上网（软路由）和数据存储（NAS）全部集成在一台机器上的玩家来说，**UPS（不间断电源）不是“有没有必要”的问题，而是“最后一块拼图”。**

### 1. 保护你的 8TB 大硬盘

你之前计划购买 **8TB 的机械硬盘**来存资料。机械硬盘最怕的就是在高速旋转读写时突然断电，这会导致：

- **物理损坏**：磁头来不及归位，可能划伤盘片，导致硬盘直接报废。
- **文件系统崩溃**：你运行的 PVE 和 NAS 系统（如飞牛）在读写数据时会有大量缓存。突然断电会导致文件索引丢失，轻则丢数据，重则整个硬盘分区变成“RAW”格式无法读取。

### 2. 保证全家“不断网”

你的系统里运行着 **iStoreOS 拨号**。如果家里闪断一下电（哪怕只有 1 秒）：

- **没 UPS**：N100 主机重启，加载 PVE、加载 iStoreOS 需要 2-3 分钟，全家断网 3 分钟，游戏断线、视频卡死。
- **有 UPS**：UPS 瞬间接管供电，你的 N100 主机毫无感觉，网络稳如泰山。

### 3. 实现“优雅关机”（PVE 的神技）

这是 UPS 最专业的功能。你需要买那种**带 USB 通讯接口**的 UPS（通常叫“NAS 专用 UPS”）：

- 当你家里停电，UPS 电池电量快耗尽时，它会通过 USB 线告诉 PVE 系统：“快撑不住了，赶紧关机！”
- PVE 接收到信号后，会自动把你的飞牛 NAS、iStoreOS 一个个安全关闭，最后关闭自己。**这能 100% 避免系统崩溃。**

---

### 📝 N100 系统该怎么选 UPS？

由于 N100 非常省电（整机满载也就 20-30W），你不需要买那种几千块的服务器级别 UPS。

**推荐规格：**

- **功率**：500VA 或 600VA（约 300-360W）就足够供电给你的 N100 + 光猫 + 两个 TP-Link 路由器了。
- **必须功能**：**带有 USB 通讯接口**（这是实现自动关机的关键）。
- **价格区间**：**¥200 - ¥350** 之间。

**推荐型号参考：**

- **山特 (SANTAK) TG-BOX 600 / 850**：外形像个插线板，非常适合放在客厅或弱电箱旁，自带 5-10 个插槽。
- **APC (施耐德) BK650-CH**：神中神，二手或新款都很稳，是 NAS 圈的标配，PVE 和各种系统对它的兼容性是最好的。
- **雷迪司 (LADIS) H500/H600**：性价比极高，也是带 USB 通讯口的。
