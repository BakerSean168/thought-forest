# JSW-header

![[JSW-header.png]]
这是一个标记为 **JSW1** 的接口。

### 1. 它是做什么的？

- **全称推测**: **J**umper **SW**itch (跳线开关) 或 **S**ide **W**itch (侧边按键板)。
- **用途**: 这是一个 **一体机 (AIO) 专用的按键板接口**。
- **背景**: 因为这块板子原本是给“显示器主机一体机”设计的。这种电脑通常会在屏幕侧面或底部有一排小按钮（开机、息屏、亮度调节等）。这个接口就是用来连接那排按钮的排线的。

### 2. 为什么会有这个接口？

它和我们在主板右下角看到的那个彩色的 **FPANEL**（前面板插针）功能其实是重复的，但物理形态不同：

- **FPANEL (彩色针)**: 给普通 DIY 机箱用的，一根一根插线。
- **JSW1 (黑色座)**: 给工厂组装用的，直接插一个 4 针的端子，效率更高，防呆不容易掉。

### 3. 这个接口的可能定义 (4 Pin)

虽然没有说明书，但根据这类工控板的惯例，这 4 根针通常包含了最基础的开关机功能，例如：

1. **PWR_SW** (开机信号)
2. **GND** (地线)
3. **PWR_LED+** (电源灯正极)
4. PWR_LED- (电源灯负极)
	
	(具体顺序需万用表测，但通常就是把开机键和灯整合在一起了)

### 4. 给你的建议

**直接忽略它。**

- **原因**: 你正在做 Home Server / NAS，使用的是普通机箱。你应该去接主板**右下角那排彩色的 FPANEL 插针**，那里才是标准的机箱开机键接法。
- **除非**: 你的 FPANEL 坏了，或者你想自己做一个特殊的“一键开关机”按钮引到桌面上，才需要研究这个接口的线序。
