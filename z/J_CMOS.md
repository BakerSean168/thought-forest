# J_CMOS 

![[J_COMS.png]]

这是一个 **CMOS 清除跳线 (Clear CMOS Header)**。

### 1. 它是做什么的？

- **功能**: 它的作用是 **重置 BIOS 设置** 到出厂默认状态。
- **别名**: 它的名字 `JCMOS1` 代表 **J**umper **CMOS**。在其他主板上可能叫 `CLR_CMOS` 或 `JBAT`。
- **工作原理**: 主板上的纽扣电池 (CR2032) 给 BIOS 芯片供电以保存你的设置（比如启动顺序、时间、密码）。这个跳线可以直接切断这部分供电或短路重置脚，让 BIOS “失忆”，恢复成出厂设置。

### 2. 物理结构

- **3 针设计**: 仔细看图，这是一排 3 根针。
- **跳线帽 (Jumper Cap)**: 目前那个黑色的塑料小方块（跳线帽）插在 **上两根针** (假设是 Pin 1-2) 上。这表示 **“正常模式” (Normal)**。
- **空出的一根**: 下面还有一根金色的针露在外面，这是用于“清除模式”的。

### 3. 什么时候需要用到它？

对于你的 N100 软路由/服务器折腾之旅，这是你的 **“后悔药”**：

- **设置错了开不了机**: 比如你在 BIOS 里乱改了内存频率、显存设置，导致黑屏无法启动。
- **忘记密码**: 如果你手贱给 BIOS 设置了密码然后忘了。
- **系统无法识别硬件**: 有时候加装新硬件不识别，清一下 CMOS 有奇效。

### 4. 如何使用？ (救砖操作指南)

如果你的机器点不亮了，按以下步骤操作：

1. **彻底断电**: 拔掉电源线。
2. **移动跳线帽**: 把那个黑色的塑料帽拔下来，插到 **下两根针** (Pin 2-3) 上。
3. **等待**: 保持这个状态 5-10 秒。
4. **复原**: **非常重要！** 必须把跳线帽插回原来的位置 (Pin 1-2)。如果不插回去，机器可能永远开不了机或者每次重启都丢失设置。
5. **开机**: 插上电源开机，BIOS 就恢复默认了。

**小贴士**: 如果你觉得拔那个小帽子太费劲（手指粗不好捏），直接把主板上那个银色的圆形大电池 (CR2032) 扣下来放电 1 分钟再装回去，效果是一样的。
