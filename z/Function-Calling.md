# Function Calling

**上级索引**：[[AI MOC]]

大语言模型只能进行简单的内容推断，对于计算推理等内容并不擅长。  
比如计算 9.11 - 9.9 时，让大语言模型直接回答很可能会是错误的。  
此时，我们可以给 大语言模型 提供一个计算函数（工具），让 大语言模型 通过这个工具来返回正确的答案。  

流程：  

1. 本地问 9.11 - 9.9，并提供可使用的工具信息
2. ai 提取 两个参数，并表示要使用工具
3. 本地获取两个参数，并使用工具计算出答案，再给 ai
4. ai 给出答案
