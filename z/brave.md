# brave 

**Brave** 是一款主打**隐私保护、安全和极速体验**的网页浏览器。它的创始人是 JavaScript 之父、Mozilla (Firefox) 的联合创始人 Brendan Eich，因此出身不凡。

以下是它的核心介绍：

### 1. 核心杀手锏：自带“盾牌” (Brave Shields)

这是 Brave 最吸引人的地方。你不需要安装任何第三方插件（如 AdBlock），Brave **默认就会拦截**网页上的：

* **商业广告：** 浏览网页、看视频（如 YouTube）通常没有广告。
* **追踪器：** 阻止广告商收集你的浏览习惯。
* **指纹脚本：** 防止网站识别你的设备特征。
* **结果：** 因为拦截了这些“垃圾”加载项，网页加载速度通常比 Chrome 快 3 到 6 倍，且更省电、省流量。

### 2. 独特的经济模型：看广告赚钱 (Brave Rewards)

Brave 试图颠覆传统的广告模式。

* **默认无广告：** 网页原本的广告被屏蔽。
* **自愿看广告：** 如果你开启“Brave Rewards”，浏览器会推送一种**保护隐私的、非侵入式**的通知广告。
* **获得报酬：** 作为交换，你会获得 **BAT (Basic Attention Token)** 加密货币代币。你可以把这些代币打赏给你喜欢的创作者（如 YouTuber、博主），或者自己提现（需绑定钱包）。

### 3. 兼容性与内核

* **基于 Chromium：** 它的内核和 Google Chrome、Microsoft Edge 是一样的。
* **无缝迁移：** 这意味着**所有的 Chrome 扩展插件**都可以在 Brave 上直接安装使用。你的书签、密码也能轻松从 Chrome 导入。

### 4. 高级隐私功能

* **Tor 模式：** 它的“隐私窗口”不仅仅是不记录历史，还集成了 **Tor (洋葱路由)** 网络，可以进一步隐藏你的真实 IP 地址，实现更高程度的匿名。

### 总结

**Brave** 是目前**Google Chrome 最强有力的隐私替代品**。
如果你厌倦了满屏的广告，或者担心大公司监控你的隐私，但又离不开 Chrome 丰富的插件生态，那么 Brave 就是为你量身定做的。
