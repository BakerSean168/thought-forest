# 每日温度 

## 题目

[739. 每日温度 - 力扣（LeetCode）](https://leetcode.cn/problems/daily-temperatures/description/)

## 题解

- [[单调栈]]

从右往左依次将温度压入单调栈，压入前，抛出所有更低温度的数据（保持单调栈单调），获取下一个升温的下标（减去当前下标，即可获得几天后升温）

```js
var dailyTemperatures = function(temperatures) {
    const n = temperatures.length;
    const ans = Array(n).fill(0);
    const st = [];
    for (let i = n-1;i>=0;i--) {
        const t = temperatures[i];
        while(st.length && t >= temperatures[st[st.length-1]]) {
            st.pop();
        }
        if (st.length) {
            ans[i] = st[st.length - 1] - i;
        }
        st.push(i);
    }
    return ans;
};
```
