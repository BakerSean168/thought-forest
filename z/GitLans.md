## 基本概念

GitLans 是 vscode 中最多下载量的 git 扩展。  

扩展了许多原生 source control 没有的功能：  
- Commit Graph
  图形化显示提交记录
- Blame
  查看修改属于谁

pro 版好像多了团队合作相关内容
