# 💻 VPS 应用 - 网络代理（Network Proxy）MOC

## 概览（Overview）

  * **这是一个什么东西？**
	  * 网络代理是一种服务或服务器，作为用户（客户端）和目标服务器（服务端）之间的**中介**。它接收客户端的请求，并将请求转发给目标服务器，再将目标服务器的响应转发回客户端。
  * **它解决什么问题？适用场景是什么？**
	  * **解决问题：** 主要解决**网络访问限制**（如地理限制、防火墙）、**隐私保护**（隐藏真实 IP 地址）、**网络加速/缓存**和**安全审计**等问题。
	  * **适用场景：**
		  * **突破网络封锁：** 访问受限的境外网站或服务。
		  * **隐私保护：** 不想暴露真实 IP 地址给目标网站。
		  * **数据安全：** 在公共网络中使用加密代理来保护数据传输。
  * **一句话总结：** **网络代理是客户端与目标服务器之间的“信使”或“跳板”，帮助用户安全、自由地访问互联网资源。**

-----

## 关键概念（Core Concepts）

  * **代理服务器 (Proxy Server)：** 运行代理服务的计算机或程序，通常由用户自己部署在 **VPS (Virtual Private Server)** 上。
  * **正向代理 (Forward Proxy)：** 客户端明确知道目标服务器地址，代理服务器位于客户端和目标服务器之间，为**客户端服务**（如上述的突破限制）。
  * **反向代理 (Reverse Proxy)：** 客户端只知道代理服务器地址，代理服务器位于目标服务器之前，为**服务器服务**（如负载均衡、缓存、安全防护，与 VPS 代理应用场景不同）。
  * **专有名词 / 缩写解释：**
	  * **VPS (Virtual Private Server)：** 虚拟专用服务器，用户远程部署代理程序的平台。
	  * **DPI (Deep Packet Inspection)：** 深度包检测，防火墙用于分析网络流量内容和特征的技术。
	  * **TUN/TAP：** 虚拟网络设备，常用于实现**全局代理**或**VPN**效果。
