# 端口无法使用问题 

## 情境

启动 java 服务时 端口为 48080，报端口被占用，换位 48081 仍然报错，检测又没有。
后面发现改为 18080 就能够正常启动。
检测端口的问题。

## 方法

### 1

使用了 ai 提供的 端口测试工具，结果没有问题

### 2

使用 已有前端项目，将端口改为 48080，结果还是没有问题

## 结论ing

是不是 Java 项目的问题，但是别人都没事
一个悬案
