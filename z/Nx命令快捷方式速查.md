# Nx命令快捷方式速查 

## Nx控制台

|**快捷键**|**功能描述**|
|---|---|
|**`?`**|切换显示/隐藏此帮助弹窗|
|**`q`** 或 **`<ctrl>+c`**|退出 TUI 界面|
|**`↑`** 或 **`k`**|向上导航/滚动任务输出|
|**`↓`** 或 **`j`**|向下导航/滚动任务输出|
|**`<ctrl>+u`**|向上翻页滚动任务输出|
|**`<ctrl>+d`**|向下翻页滚动任务输出|
|**`/`**|根据搜索关键词过滤任务|
|**`<esc>`**|清除过滤器 / 将焦点返回至任务列表|
|**`<enter>`**|打开并聚焦到该任务的终端|
|**`<space>`**|快速切换单个输出窗格的显示状态|
|**`b`**|切换任务列表的可见性（显示/隐藏）|
|**`m`**|在垂直和水平布局之间切换（默认为自动）|
|**`1`**|将任务固定在输出窗格 1 中显示|
|**`2`**|将任务固定在输出窗格 2 中显示|
|**`<tab>`**|在任务列表与输出窗格 1、2 之间切换焦点|
|**`c`**|复制当前选中的输出内容到剪贴板|
|**`i`**|当持续性任务处于焦点时，与其进行交互|
|**`<ctrl>+z`**|停止与持续性任务的交互|
