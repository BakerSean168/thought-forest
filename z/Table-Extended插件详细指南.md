# Table Extended 插件详细指南

> **核心功能**：增强表格编辑、快速操作、公式支持、列排序  
> **难度级别**：⭐⭐  
> **推荐指数**：⭐⭐⭐⭐  
> **必装程度**：推荐（表格管理增强）

---

## 功能概览

### 核心特性

| 功能 | 说明 | 优势 |
|-----|------|------|
| **快速编辑** | 右键菜单快速操作 | 减少手动操作 |
| **行列管理** | 插入/删除行列 | 灵活调整 |
| **列排序** | 按列排序表格 | 数据重组 |
| **行移动** | 拖拽移动行位置 | 直观操作 |
| **快速导航** | Tab 键快速移动单元格 | 提高效率 |
| **对齐方式** | 左/中/右对齐 | 美化表格 |
| **公式计算** | 简单计算功能 | 动态数据 |

### 支持的操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 编辑单元格 | 双击 | 进入编辑模式 |
| 移动到下一个 | Tab | 右移单元格 |
| 移动到上一个 | Shift+Tab | 左移单元格 |
| 新增行 | 下箭头（在最后一行） | 自动添加行 |
| 右键菜单 | 右键 | 显示快速操作 |

## 安装与配置

### 安装

1. Settings → Community plugins → Browse
2. 搜索 "Table Extended"
3. 安装并启用

### 基本配置

```yaml
Settings → Table Extended

## 编辑设置
Enable Table Editor: true             # 启用表格编辑器
Show Row Numbers: true                # 显示行号
Show Table Grid: true                 # 显示表格网格

## 快捷键设置
Ctrl+Shift+A: 添加行
Ctrl+Shift+D: 删除行
Ctrl+Shift+L: 左对齐
Ctrl+Shift+C: 居中对齐
```

## 快速开始

### 创建基础表格

**Markdown 表格语法**

```markdown
| 标题 1 | 标题 2 | 标题 3 |
|--------|--------|--------|
| 数据 1 | 数据 2 | 数据 3 |
| 数据 4 | 数据 5 | 数据 6 |
```

### 快速编辑操作

| 操作 | 方法 |
|------|------|
| 编辑单元格 | 双击单元格 |
| 插入行 | 右键 → Insert Row Above/Below |
| 删除行 | 右键 → Delete Row |
| 插入列 | 右键 → Insert Column Left/Right |
| 删除列 | 右键 → Delete Column |
| 排序表格 | 点击列头 → Sort |
| 对齐 | 右键 → Align Left/Center/Right |

**原始数据表**


## 常见问题

### Q1：如何快速添加多行？

**方法 1：逐行添加**

```
右键最后一行 → Insert Row Below
重复操作添加多行
```

**方法 2：复制粘贴**

```
1. 选中一行
2. Ctrl+C 复制
3. 定位到目标行
4. Ctrl+V 粘贴
5. 修改数据
```

### Q2：如何重新排列列的顺序？

**步骤**

```
1. 右键列头
2. Move Column Left / Move Column Right
3. 逐步调整到目标位置

或手动删除列，重新插入
```

### Q3：表格格式乱了怎么办？

**重新格式化**

```
右键表格 → Format Table
自动调整对齐和格式
```

### Q4：如何给列指定宽度？

**方法**

```markdown
标准 Markdown 无法指定宽度
但可以通过添加空格调整视觉宽度：

| 名称 | 描述 |
|-----------|--------------------------|
| 简短 | 一个较长的描述说明 |
```

### Q5：如何导出表格到其他格式？

**导出步骤**

```yaml
方法 1：复制为 HTML
右键 → Copy as HTML → 粘贴到其他工具

方法 2：导出为 CSV
Better Export PDF 插件可以导出表格

方法 3：手动复制
选中表格内容 → Ctrl+C → 粘贴到 Excel
```

## 高级技巧

### 1. 使用公式计算

```markdown
| 商品 | 单价 | 数量 | 合计 |
|------|------|------|------|
| A | 100 | 5 | 500 |
| B | 200 | 3 | 600 |
| C | 150 | 2 | 300 |
| 总计 | | | 1400 |

最后一行的合计可以手动计算或使用外部工具
```

### 2. 美化表格显示

**使用对齐**

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:--------|:----------:|----------:|
| 左 | 中 | 右 |
```

**右键操作**

```
1. 选中列
2. 右键 → Align Center
3. 表格更美观
```

### 3. 合并与分组

```markdown
虽然 Markdown 表格不支持直接合并
但可以用分组标记：

| 分类 | 项目 | 值 |
|------|------|------|
| A | A1 | 100 |
| A | A2 | 150 |
| B | B1 | 200 |
| B | B2 | 250 |
```

## 最佳实践

1. **保持表格简洁**
   - 列数不超过 8
   - 行数适度（超过 20 行考虑分表）
   - 避免嵌套复杂内容

2. **规范数据格式**
   - 日期统一格式
   - 数字右对齐
   - 文本左对齐

3. **定期清理**
   - 删除过期数据
   - 归档历史表格
   - 保持表格活跃

4. **配合其他工具**
   - 复杂计算用 Excel
   - 简单表格用 Markdown
   - 查询用 Dataview
