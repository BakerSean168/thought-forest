# 新笔记本到手该做哪些事 

## 基础概念

新机到手必得小心翼翼

## 事先准备

U盘（图吧工具箱）
## 验机流程

### 1. 包转检查（破损、配置单）

office 贴纸、配置单

### 2. 外观查看

电源插头(头有没有歪)
适配器

笔记本本体拿起来在耳边晃动，判断有无螺丝掉落（有的话开机容易烧坏机器）
A 壳有无掉漆
接口有无变形、插拔痕迹
贴纸
开合屏幕转轴阻尼
屏幕有无碎裂、漏液
键盘卡壳、脱落
触控板触点返回

### 3. 开机设置

开机3次，查看运输模式是否还在

插上电源开机，系统自检不能合盖

**跳过联网激活**：
```
shift + F10

oobe\bypassNRO.cmd
```

重启后选择 我没有internet 选项

**进入系统**

检查激活：
设置-》系统-》激活
无法连接代表是没有激活的

### 4. 电池报告

查看电池报告：  
cmd -》 `powercfg/batteryreport`
在浏览器打开地址

### 5. 硬件核对

打开图吧工具箱

- 硬件配置核对
- 屏幕检测
	- 坏点与屏幕光测试
- 键盘检测
	- Keyboard Test Utility
- 硬盘
	- DistInfo
	- 通电事件怎么着也不能是 0 h，可能是被换过硬盘
- [[烤机]]

打开性能模式

Fn hotkeys and OSD（性能模式+独显直连）
