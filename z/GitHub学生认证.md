## 基础条件

- 校园邮箱 + 账号
	1. 在已有账号中添加校园邮箱
	2. 直接用校园邮箱注册账号

## 操作步骤

### 创建账号

账号、密码、邮箱、名称、邮箱验证

## two-factor authentication

用于多重认证，直接使用认证令牌模式（MFA，阿里云、华为云 app 自带，或者 类似软件：微软的 Authenticator），

`Settings/Access/Password and authentication`

### **补充个人信息**


## 开始申请

**入口**
直接封面跳转屏有 `Join GitHub Education` 到 `Access/Billing and licensing/Education benefits` 中有 `Start an application` 按钮
![[Pasted image 20251013184052.png]]
![[Pasted image 20251013184128.png]]

验证材料上传，正常应该能直接 upload file

![[c8d8d69120e8859ab2a95973f5ae47ae.png]]
填表、提交信息
- 需要学生卡或者学生证这种能信息，上传
- 需要共享地址，如果远离学校需要说明原因，提供证明

## 坑点

1. 要完善个人信息，否则在校园里也会不允许上传文件，并且有「你为什么不在学校的问题」
2. firefox浏览器中点击 submit application 会卡住
3. 上传的图片要小于 1M，记得压缩，[图片免费压缩](https://docsmall.com/image-compress)
