# ESMAScript操作符

以下是 ECMAScript（ES）最新版本（ES2023）中所有操作符的总结。ES2023 主要继承了之前版本的操作符，没有新增核心操作符（ES2021 的逻辑赋值操作符 ||=、&&=、??= 是最近的重要补充）。我按类别整理成表格，包括操作符、名称、说明和简单样例。所有操作符都是通用的 JavaScript 语法。

## 基础概念
### 1. 算术操作符

| 操作符 | 名称 | 说明 | 样例 |
|--------|------|------|------|
| `+` | 加法 | 数字相加，或字符串拼接 | `5 + 3` → 8; `"a" + "b"` → "ab" |
| `-` | 减法 | 数字相减 | `5 - 3` → 2 |
| `*` | 乘法 | 数字相乘 | `5 * 3` → 15 |
| `/` | 除法 | 数字相除 | `6 / 3` → 2 |
| `%` | 取余 | 返回除法余数 | `7 % 3` → 1 |
| `++` | 自增 | 变量值加 1（前缀/后缀） | `let x = 5; x++` → x = 6 |
| `--` | 自减 | 变量值减 1（前缀/后缀） | `let x = 5; --x` → x = 4 |
| `**` | 幂运算 (ES2016) | 计算幂 | `2 ** 3` → 8 |

### 2. 比较操作符

| 操作符   | 名称    | 说明         | 样例                  |
| ----- | ----- | ---------- | ------------------- |
| `==`  | 相等    | 值相等（类型转换）  | `5 == "5"` → true   |
| `===` | 严格相等  | 值和类型都相等    | `5 === "5"` → false |
| `!=`  | 不相等   | 值不相等（类型转换） | `5 != "5"` → false  |
| `!==` | 严格不相等 | 值或类型不相等    | `5 !== "5"` → true  |
| `>`   | 大于    | 左值大于右值     | `5 > 3` → true      |
| `<`   | 小于    | 左值小于右值     | `3 < 5` → true      |
| `>=`  | 大于等于  | 左值大于或等于右值  | `5 >= 5` → true     |
| `<=`  | 小于等于  | 左值小于或等于右值  | `3 <= 5` → true     |

### 3. 逻辑操作符

| 操作符 | 名称 | 说明 | 样例 |
|--------|------|------|------|
| `&&` | 逻辑与 | 两个操作数都为真时返回真 | `true && false` → false |
| `\|\|` | 逻辑或 | 任一操作数为真时返回真 | `true \|\| false` → true |
| `!` | 逻辑非 | 取反布尔值 | `!true` → false |

### 4. 赋值操作符

| 操作符 | 名称 | 说明 | 样例 |
|--------|------|------|------|
| `=` | 赋值 | 将右值赋给左值 | `let x = 5;` |
| `+=` | 加赋值 | 左值 += 右值，等价于左值 = 左值 + 右值 | `x += 3;` (x = 8) |
| `-=` | 减赋值 | 左值 -= 右值 | `x -= 2;` (x = 6) |
| `*=` | 乘赋值 | 左值 *= 右值 | `x *= 2;` (x = 10) |
| `/=` | 除赋值 | 左值 /= 右值 | `x /= 2;` (x = 2.5) |
| `%=` | 取余赋值 | 左值 %= 右值 | `x %= 3;` (x = 2) |
| `**=` | 幂赋值 (ES2016) | 左值 **= 右值 | `x **= 2;` (x = 25) |
| `<<=` | 左移赋值 | 左值 <<= 右值 | `x <<= 1;` (x = 10) |
| `>>=` | 右移赋值 | 左值 >>= 右值 | `x >>= 1;` (x = 2) |
| `>>>=` | 无符号右移赋值 | 左值 >>>= 右值 | `x >>>= 1;` (x = 2) |
| `&=` | 位与赋值 | 左值 &= 右值 | `x &= 3;` (x = 0) |
| `\|=` | 位或赋值 | 左值 \|= 右值 | `x \|= 3;` (x = 7) |
| `^=` | 位异或赋值 | 左值 ^= 右值 | `x ^= 3;` (x = 6) |
| `&&=` | 逻辑与赋值 (ES2021) | 左值 &&= 右值 | `x &&= 3;` (如果 x 为真，x = 3) |
| `\|\|=` | 逻辑或赋值 (ES2021) | 左值 \|\|= 右值 | `x \|\|= 3;` (如果 x 为假，x = 3) |
| `??=` | 空值合并赋值 (ES2021) | 左值 ??= 右值 | `x ??= 3;` (如果 x 为 null/undefined，x = 3) |

### 5. 位操作符

| 操作符   | 名称    | 说明       | 样例                                 |
| ----- | ----- | -------- | ---------------------------------- |
| `&`   | 位与    | 按位与运算    | `5 & 3` → 1 (二进制: 101 & 011 = 001) |
| `\|`  | 位或    | 按位或运算    | `5 \| 3` → 7 (101 \| 011 = 111)    |
| `^`   | 位异或   | 按位异或运算   | `5 ^ 3` → 6 (101 ^ 011 = 110)      |
| `~`   | 位非    | 按位取反     | `~5` → -6                          |
| `<<`  | 左移    | 左移位      | `5 << 1` → 10 (101 << 1 = 1010)    |
| `>>`  | 右移    | 右移位（带符号） | `5 >> 1` → 2 (101 >> 1 = 10)       |
| `>>>` | 无符号右移 | 右移位（无符号） | `-5 >>> 1` → 2147483645            |

### 6. 其他操作符

| 操作符 | 名称 | 说明 | 样例 |
|--------|------|------|------|
| `?:` | 条件（三元） | 条件 ? 值1 : 值2 | `5 > 3 ? "yes" : "no"` → "yes" |
| `,` | 逗号 | 顺序执行，返回最后一个值 | `(1, 2, 3)` → 3 |
| `...` | 展开/剩余 (ES2015) | 展开数组/对象，或收集剩余参数 | `[...arr]` (展开); `function f(...args)` (剩余) |
| `?.` | 可选链 (ES2020) | 安全访问属性，避免 null/undefined 错误 | `obj?.prop` |
| `??` | 空值合并 (ES2020) | 返回左值或右值（左值为 null/undefined 时） | `null ?? "default"` → "default" |
| `delete` | 删除 | 删除对象属性 | `delete obj.prop;` |
| `typeof` | 类型检查 | 返回值的类型字符串 | `typeof 5` → "number" |
| `instanceof` | 实例检查 | 检查对象是否为构造函数的实例 | `arr instanceof Array` → true |
| `in` | 属性存在 | 检查属性是否存在于对象中 | `"prop" in obj` → true |
| `new` | 实例化 | 创建对象实例 | `new Date()` |
| `void` | 空值 | 返回 undefined | `void 0` → undefined |

**注意**：
- 样例中的 `x` 假设初始值为 5。
- 位操作符主要用于低级编程，现代 JS 中较少使用。
- ES2023 没有新增操作符，但强化了现有语法（如私有字段 `#`，但这不是操作符）。
- 如果需要更详细的解释或特定版本的变更，请提供更多上下文。参考：[MDN 操作符文档](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Operators)。

## 经验总结


JavaScript 中的 `==`（宽松相等）和 `===`（严格相等）是两种不同的比较运算符，核心区别在于 是否进行类型转换 以及 对特殊值的处理规则
