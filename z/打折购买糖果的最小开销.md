# 打折购买糖果的最小开销 

## 题目

[2144. 打折购买糖果的最小开销 - 力扣（LeetCode）](https://leetcode.cn/problems/minimum-cost-of-buying-candies-with-discount/description/)

## 题解

### [[贪心算法#单序列配对]]

```js
var minimumCost = function(cost) {
    cost = cost.sort((a,b) => b - a);
    let index = 0;
    let res = 0;
    for (const c of cost) {
        index++
        if (index % 3 === 0) {
            continue
        }
        res += c;
    }
    return res;
};
```
