# Umi-OCR 

该应用基于 RapidOCR 引擎，最大的特点是无需联网即可运行，极大地保护了用户的隐私安全。相比标准的 Paddle 引擎版本，Rapid 版体积更小、内存占用更低，且对旧版系统（如 Windows 7）以及缺乏 AVX 指令集的低端处理器有更好的支持，确保在各类硬件环境下都能稳定运行。

- **核心功能：** 支持截图 OCR、批量导入图片识别、PDF 文档识别（可转为双层可搜索 PDF）以及二维码扫描与生成。
- **技术亮点：** 内置高性能离线引擎，提供排版解析功能（自动合并自然段），并支持通过 HTTP 接口或命令行进行自动化调用。
- **适用场景：** 适用于需要处理敏感文档、批量提取图片文字、在无网环境办公以及使用老旧电脑进行高效办公的用户。
