## 恢复出厂设置



打了 400-700-1000，技术人员说设置里的恢复出厂设置和 alt+F10 的恢复出厂设置是一样的
问了 gemini 又说是不一样的：

| **特性**   | **Alt + F10 (原厂 D2D)**        | **设置 - 本地重装**   |
| -------- | ----------------------------- | --------------- |
| **驱动程序** | **原厂特调驱动**，完美兼容硬件             | 通用驱动或当前系统残留驱动   |
| **厂商工具** | **自带 NitroSense** 和 Acer 控制中心 | 可能丢失，需手动重装      |
| **系统版本** | 回到购机时的原始版本                    | 保持当前的系统版本       |
| **推荐人群** | **追求极致稳定和原厂优化的用户**            | 只是想快速修复系统小问题的用户 |

还是先选择 Alt+F10 Restore Factory Settings
直接把全部都格式化了。
