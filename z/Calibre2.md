# Calibre2 

**Calibre**（你看到的 `Calibre2` 很可能是它的默认安装文件夹名称）是全球最著名、功能最强大的**免费开源电子书管理软件**。

对于喜欢阅读电子书（特别是拥有 Kindle、Kobo 等电子书阅读器）的人来说，它是绝对的“神器”和必装软件。

以下是它的核心介绍：

### 1. 核心定位：电子书的“iTunes”

它不仅仅是一个阅读器，而是一个**全能的电子书图书馆**。

* **管理中心：** 你可以把电脑里所有的 EPUB, MOBI, PDF, AZW3 等各种格式的电子书都拖进去，它会帮你自动读取封面、作者、简介，并整齐地排列好。
* **元数据编辑：** 它可以自动从互联网（如 Amazon、豆瓣）下载书籍的详细信息、标签和高清封面，帮你把杂乱的文件整理得井井有条。

### 2. 杀手级功能：格式转换

这是 Calibre 最被推崇的功能，被称为电子书界的“万能格式工厂”。

* **解决痛点：** 比如你下载了一个 `.epub` 格式的书，但你的 Kindle 只能看 `.mobi` 或 `.azw3`。
* **一键转换：** Calibre 可以完美地将一种格式转换为另一种格式，并且在转换过程中还能自动优化排版、目录和字体大小。

### 3. 设备同步

* **发送到设备：** 当你把 Kindle 或其他阅读器连上电脑，Calibre 会自动识别。点击“发送到设备”，它会自动把书转换成该设备支持的最佳格式并传输进去。
* **邮件推送：** 它可以配置 SMTP，一键把书通过邮件推送到你的 Kindle 账户，实现无线传输。

### 4. 进阶功能

* **插件系统：** Calibre 拥有强大的插件生态。最著名的是 **DeDRM** 插件，配合使用后，它可以移除购买的电子书（如 Amazon Kindle 书店）的版权保护（DRM），让你真正拥有这本书，可以在任何设备上阅读。
* **电子书编辑：** 内置了专业的编辑器，懂 HTML/CSS 的用户可以深入修改电子书的内部代码，修正错别字或调整排版。

### 总结

**Calibre** 是一款**功能极度强大但界面略显复古**的软件。

* 如果你只在手机上看书，可能用不上它。
* 如果你有 **Kindle/Kobo 阅读器**，或者你有**囤积和整理电子书**的习惯，它是目前地球上最好的选择，没有之一。
