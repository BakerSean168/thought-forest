# 电脑黑屏重启问题 

## 说明

电脑突然黑屏然后重启了

## 探查

### 检查系统日志

主要通过**事件查看器**筛选关键日志：
1. **打开事件查看器**
	有两种便捷方式，一是按下`Win+R`打开运行窗口，输入`eventvwr.msc`后回车；二是右键点击 “此电脑”，选择 “管理”，在弹出的窗口中找到 “事件查看器”。
2. **定位并筛选系统日志**
	在左侧面板展开 “Windows 日志”，点击 “系统”。接着点击右侧 “筛选当前日志”，事件级别勾选 “关键”“错误”“警告”，再输入关键事件 ID 可精准匹配问题：输入 41 对应电源故障或意外重启；输入 6008 对应系统意外关闭；输入 1074 对应程序或用户触发的重启。
3. **解读关键日志**
	双击筛选出的日志条目查看详情，若来源显示`Kernel-Power`，大概率是电源或硬件相关故障；来源为`nvlddmkm`则可能是 NVIDIA 显卡驱动异常；来源为`Resource-Exhaustion-Detector`，可能是虚拟内存耗尽或软件占用资源过高导致黑屏重启。
4. **PowerShell 快速查询（可选）**
	按下`Win+X`选择 “Windows PowerShell（管理员）”，输入命令`Get-EventLog -LogName System | Where-Object { $_.EventID -eq 6008 } | Select-Object -First 1`，可快速调出最近一次异常关机的日志记录。

筛选 41 事件id，确定具体时间后，查看前后的错误信息，发现可能是 volmgr（硬盘问题）或者 Hyper-V（虚拟化问题）
