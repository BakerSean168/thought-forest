# 直接邀请朋友修改仓库代码 

## 实战经验

### ✅ **终极简化方案（需朋友配合）**
#### ⚡ 前提条件：让朋友在仓库设置中为你添加 **`Collaborator`（协作成员）权限**
1. 朋友操作：
仓库 → `Settings` → `Collaborators` → 输入你的GitHub账号 → 点击 **`Add collaborator`**
→ 你邮箱会收到权限邀请 → 接受邀请。

#### ⚡ 获得权限后操作（3步直接修改）：
1. **克隆原仓库**
```bash
git clone https://github.com/朋友的账号/仓库名.git
cd 仓库名
```
2. **修改代码并提交**
```bash
git checkout -b your-edit# 新建分支（名称自定）
# 用VS Code等工具直接修改文件 → 保存
git add .
git commit -m "你的修改说明"
```
3. **直接推送到原仓库**
```bash
git push origin your-edit# 分支直达朋友仓库
```
→ 朋友在仓库的 `Pull requests` 标签页即可看到你的修改，一键合并。
