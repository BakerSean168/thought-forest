# Outline 插件详细指南

> **核心功能**：文档导航、大纲视图、快速跳转  
> **难度级别**：⭐⭐  
> **推荐指数**：⭐⭐⭐⭐⭐  
> **必装程度**：强烈推荐（长文档必备）

---

## 功能概览

### 核心特性

| 功能 | 说明 | 优势 |
|-----|------|------|
| **大纲面板** | 右侧边栏显示笔记大纲 | 快速浏览文档结构 |
| **快速跳转** | 点击标题直接跳转 | 大文档快速导航 |
| **层级显示** | 标题层级清晰展示 | 理解文档逻辑 |
| **实时更新** | 编辑时大纲同步更新 | 保持同步 |
| **折叠展开** | 可折叠大纲树 | 减少屏幕占用 |
| **搜索位置** | 显示在文档哪个位置 | 上下文感知 |

### Outline 界面示例

```
当前笔记：我的学习笔记.md

├─ # 学习笔记
│  ├─ ## 概念理解
│  │  ├─ ### 定义
│  │  └─ ### 原理
│  ├─ ## 实践应用
│  │  ├─ ### 示例 1
│  │  └─ ### 示例 2
│  └─ ## 总结
```

## 安装与配置

### 安装

1. Settings → Community plugins → Browse
2. 搜索 "Outline"
3. 安装并启用（通常是 Obsidian 内置功能）

### 基本配置

```yaml
Settings → Outline

## 显示选项
Show Outline: true                 # 显示大纲面板
Follow Active Heading: true        # 跟随当前编辑位置
Show Heading Icon: true            # 显示标题图标
Collapse Level: 2                  # 默认展开到第几层

## 外观选项
Font Size: 13px                    # 字体大小
Indentation: 16px                  # 缩进大小
```

## 快速开始

### 打开大纲面板

**方法 1：快捷键**

```yaml
默认快捷键：
Ctrl+Shift+O（Windows）
Cmd+Shift+O（Mac）
```

**方法 2：菜单打开**

```
View → Toggle Outline
或右键笔记 → Open Outline
```

### 基本操作

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 展开大纲 | 点击 ▶ | - |
| 跳转到标题 | 点击标题名 | - |
| 收起大纲 | 点击 ▼ | - |
| 关闭面板 | 点击 ✕ | Ctrl+Shift+O |

## 常见问题

### Q1：大纲面板不显示怎么办？

**解决方案**

```yaml
1. 确认大纲面板已启用
   View → Toggle Outline

2. 确认快捷键有效
   Settings → Hotkeys → 搜索 "Outline"

3. 检查笔记是否有标题
   Outline 只显示有标题的笔记

4. 重启 Obsidian
   关闭并重新打开
```

### Q2：为什么大纲显示不完整？

**原因与解决**

```yaml
可能原因 1：标题层级不规范
✅ 正确：# H1, ## H2, ### H3（递进）
❌ 错误：# H1, ### H3（跳级）

可能原因 2：大纲面板宽度太小
→ 拖拽分割线扩大面板

可能原因 3：标题不规范
✅ 正确：使用 # ## ### 标记
❌ 错误：使用 **粗体** 代替标题
```

### Q3：如何快速跳转到特定标题？

**方法**

```yaml
1. 快捷跳转：
   Settings → Hotkeys → 搜索 "Jump to"
   配置快捷键

2. 使用搜索：
   Ctrl+F 搜索标题文本
   Ctrl+L 搜索并跳转

3. Outline 中直接点击
   最直观的方法
```

### Q4：大纲面板占用空间太大？

**优化方案**

```yaml
1. 折叠大纲
   点击 ▼ 符号

2. 自动隐藏
   Settings → Outline → Auto Hide: true

3. 按需打开
   需要时 Ctrl+Shift+O 打开
   不需要时立即关闭

4. 调整面板宽度
   拖拽分割线缩小面板
```

### Q5：Outline 更新延迟？

**性能优化**

```yaml
1. 检查标题数量
   标题过多（>100）会影响性能

2. 禁用实时高亮
   Settings → Outline → Follow Active Heading: false

3. 重启 Obsidian
   清除缓存

4. 检查其他插件
   Outline 与其他导航插件可能冲突
```

## 最佳实践

### 1. 规范标题结构

**✅ 推荐结构**

```markdown
# 主标题（H1，一篇笔记只有一个）

## 章节标题（H2，主要分章节）

### 小节标题（H3，详细细节）

#### 要点标题（H4，不推荐过深）
```

**❌ 避免**

```markdown
# H1
# H1（重复）
### H3（跳级，跳过 H2）
```

### 2. 优化大纲显示

**设置合理的折叠深度**

```yaml
Settings → Outline
Collapse Level: 2  # 默认显示到 H2，可点击展开

这样避免一开始大纲过长
```

### 3. 添加导航链接

**在笔记顶部添加快速导航**

```markdown
---
# 内容导航
- [[#基础理论]]
- [[#模型架构]]
- [[#实战应用]]
---

# 主标题

## 基础理论
...

## 模型架构
...

## 实战应用
...
```

### 4. 搭配目录生成插件

```markdown
使用 Outline 规划结构后，
可配合自动目录生成插件如 "Auto TOC" 
自动生成文档目录
```

## 与其他插件配合

### Outline + Hover Editor

```
1. 在 Outline 中点击标题
2. 同时在 Hover Editor 中查看相关笔记
3. 完整的导航和参考体验
```

### Outline + Headings Shortcuts

```markdown
快速创建标题：
- Ctrl+Alt+1 → # H1
- Ctrl+Alt+2 → ## H2
- Ctrl+Alt+3 → ### H3

创建好标题后，Outline 自动更新
```

### Outline + Templater

```markdown
在模板中自动生成目录：
<%* 
// 生成目录
-%>
## 目录
- [[#简介]]
- [[#主要内容]]
- [[#总结]]

# 简介
...
```
