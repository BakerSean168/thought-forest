# F_PANEL_header 

![[F_PANEL_header.png]]

这是一个非常标准且标记清晰的 **FPANEL1 接口**，也就是 **机箱前面板跳线接口**。

这是你装机过程中**最重要**的一个接口。你需要把机箱上的开机键、重启键、电源灯等线插在这里，电脑才能通过按钮正常开关机。

### 1. 针脚定义图解 (根据丝印)

这块主板非常贴心地使用了**彩色底座**和**文字标注**来区分针脚，比那些全黑的盲插主板友好太多了。

根据图片上的丝印，接线逻辑是 **“左右分功能，上下分行”**：

- **第一排 (Top Row)**：
	- **左侧 (绿色底座)**: `PWR_LED` = **电源指示灯**。
		- 注意看上面的 `+` 和 `-` 标记：**最左边那根针是正极 (+)**。接线时，把机箱 `POWER LED` 线的正极（通常是彩色线）插在左边，负极（通常是白/黑线）插在右边。
	- **右侧 (黄/黑色底座)**: `PW_ON` = **电源开关 (Power Switch)**。
		- 这是**核心**。把机箱上的 `POWER SW` 线插在这里。开关线不分正负极，随便插都能用。短接这两根针就是开机。
- **第二排 (Bottom Row)**：
	- **左侧 (红色底座)**: `HD_LED` = **硬盘指示灯**。
		- 通常左边是正极。接机箱的 `HDD LED` 线。插反了灯不会亮，但不会坏，如果不亮调换一下方向即可。
	- **右侧 (蓝色底座)**: `RST` = **重启开关 (Reset)**。
		- 接机箱的 `RESET SW` 线。同样不分正负极，随便插。

### 2. 接线口诀

为了方便记忆，你可以按照颜色区域来插：

- **绿色** -> 电源灯 (Power LED)
- **红色** -> 硬盘灯 (HDD LED)
- **黄/黑** -> **开机键 (Power Switch)** [最重要]
- **蓝色** -> 重启键 (Reset Switch)

### 3. 对于你的 N100 家庭服务器的建议

- **必接**: `PW_ON` (右上角)。哪怕你设置了“来电自启”，也建议接上开机键，方便强制关机或调试。
- **可选**: `HD_LED` (左下角)。如果你做 NAS，接上这个灯可以看到硬盘有没有在读写（闪烁说明在忙），对于判断死机还是忙碌很有用。
- **极简主义**: 如果你是那种把主板扔在角落里不接机箱的“裸奔”玩家，你需要开机时，用螺丝刀轻轻碰一下右上角 `PW_ON` 的那两根金属针，主板就会启动。
