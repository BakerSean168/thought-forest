# 重新分装苹果 

## 题目

[3074. 重新分装苹果 - 力扣（LeetCode）](https://leetcode.cn/problems/apple-redistribution-into-boxes/description/)

## 题解

### [[贪心算法#贪心算法详解#从最小/最大开始贪心]]

```js
var minimumBoxes = function (apple, capacity) {
    let res = 0;
    let sum = apple.reduce((acc, cur) =>
        acc + cur, 0
    )
    capacity = capacity.sort((a, b) => b - a)
    for (const cap of capacity) {
        sum -= cap;
        res++;
        if (sum <= 0) {
            return res;
        }
    }
    return res;
};
```
